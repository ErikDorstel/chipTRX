# chipTRX
ESP32 and DRA818V based VHF transceiver
#### Features
* uses DRA818V as VHF transceiver
* Frequency Range: 134 to 174MHz
* Sensitivity: -122dBm
* Output power: +27/30dBm
* CTCSS / CDCSS codes
#### GPIO
* GPIO 16 - UARTRX
* GPIO 17 - UARTTX
* GPIO 18 - SquelchIO
* GPIO 19 - PTTIO
* GPIO 23 - SleepIO
* GPIO 27 - BoostIO
* GPIO 25 - analogOut
* GPIO 32 - analogIn
